# AI_Chatbot_customUI
A modern, minimal, and responsive AI chatbot interface built using **React**, integrated with **Google OAuth** for secure sign-in and **Gemini LLM** for smart conversation handling.
---

## Features

- **Google OAuth Authentication** – Secure and seamless login
- **Gemini LLM Integration** – Intelligent, context-aware conversations
- **Clean Chat Interface** – Simple, user-friendly, and dark-themed
- **Chat History** – Persistent and scrollable conversation threads
- **Token Display** – Real-time input/output token usage
- **Feedback Buttons** – Rate AI responses instantly

---
## Tech Stack

- **Frontend**: React, CSS
- **Authentication**: Firebase or Google Identity Platform (OAuth 2.0)
- **AI Backend**: Gemini Language Model API
- **State Management**: useState, useEffect, Context API (or Redux)
- **Routing & Auth Guarding**: React Router
